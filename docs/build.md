### Variables and classes

Use the prefix `fe-` to define a variable or a class. `fe-` refers to the initials of the framework, as in "Feather".
#### Examples

```scss
// Variables
$fe-primary-color: #3498db;
$fe-font-size-large: 18px;
$fe-margin-small: 5px;

// Classes
.fe-button {
    padding: 10px;
}

.fe-container {
    max-width: 1200px;
}

.fe-header {
    background-color: #f8f9fa;
}
```
